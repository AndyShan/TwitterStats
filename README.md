#TwitterStats
## TECH

 - python
 - flask
 - mongodb
 - sigma.js

## MODEL

 - tf-idf
 - LR

## FUNCTION

```api_twitter.py``` 用于获取twitter上的一些数据，具体有：

 - 指定用户的账户信息
 - 指定用户的推文
 - 指定用户的关注者id
 - 指定用户关注的人的id
 - 指定用户的1hop用户
 - 指定用户的1hop用户关系
 - 指定用户的2hop用户
 - 指定用户的2hop用户关系

此外，目前还有以下功能：

 - 分析指定用户发送推文的关键词
